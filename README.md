# Используемый датасет

## Plastic Bottle Image Dataset

https://www.kaggle.com/datasets/siddharthkumarsah/plastic-bottles-image-dataset


Набор данных содержит изображения пластиковых бутылок с размеченными bounding box в формате YOLO.

### Структура датасета

```text
dataset/
├── train/
│   ├── images/
│   └── labels/
├── valid/
│   ├── images/
│   └── labels/
├── test/
│   ├── images/
│   └── labels/
```

### Размер выборки

* Train: 1104 изображений
* Validation: 606 изображений
* Test: 159 изображений
